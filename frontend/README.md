# Frontend — React + TypeScript + Vite

> **Semana 4 / rúbrica:** El login y registro se implementan en esta SPA (React + Vite), no en Next.js; el comportamiento es equivalente (formularios + JWT + Redux). Ver el [`README.md`](../README.md) en la raíz del proyecto para la validación completa (Semana 4 y Semana 5).

## Scripts

- `npm run dev` — servidor de desarrollo
- `npm run build` — compilación de producción

## Stack

React 19, Vite, Redux Toolkit, Tailwind CSS.
